The RequestSettings type exposes the following members.

Properties

  Name Description
Application
returns the application name
AuthSubToken
returns the authsub token to use for a webapplication scenario
AutoPaging
AutoPaging specifies if a feed iterator should return to the server to fetch more data automatically. If set to false, a loop over feed.Entries will stop when the currently fetched set of data reaches it's end. This is false by default.
ClientLoginHandler
ClientLoginHandler - this is the URI that is used to retrieve a client login authentication token
ConsumerKey
returns the ConsumerKey
ConsumerSecret
returns the ConsumerSecret
Credentials
returns the Credentials in case of a client login scenario
Maximum
the Maximum specifies how many entries should be retrieved in total. This works together with . If set, AutoPaging of a feed will stop when the specified amount of entries was iterated over. If Maximum is smaller than PageSize (), an exception is thrown. The default is -1 (ignored).
OAuthDomain
returns the OAuth Domain
OAuthUser
returns the OAuth User
PageSize
the pagesize specifies how many entries should be retrieved per call. If not set, the server default will be used. Set it either to -1 (for default) or any value > 0 to set the pagesize to something the server should honor. Note, that this set's the max-results parameter on the query, and the server is free to ignore that and give you less entries than you have requested.
PrivateKey
returns the private key used for authsub authentication
Timeout
gets and sets the Timeout property used for the created HTTPRequestObject in milliseconds. if you set it to -1 it will stick with the default of the HTTPRequestObject. From MSDN: The number of milliseconds to wait before the request times out. The default is 100,000 milliseconds (100 seconds).
Token
returns the Token for oAuth
TokenSecret
returns the TokenSecret for oAuth
UseSSL
gets and sets the SSL property used for the created HTTPRequestObject. If true, all requests will use https The default is false.
