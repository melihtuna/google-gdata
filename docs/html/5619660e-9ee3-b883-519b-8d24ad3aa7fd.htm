PageSize
the pagesize specifies how many entries should be retrieved per call. If not set, the server default will be used. Set it either to -1 (for default) or any value > 0 to set the pagesize to something the server should honor. Note, that this set's the max-results parameter on the query, and the server is free to ignore that and give you less entries than you have requested.
(Inherited from RequestSettings.)
